# RAG Chatbot (HuggingFace Embeddings + FAISS + Ollama)

This is a **Retrieval Augmented Generation (RAG) chatbot** that can handle pdfs and docx files.

It is built using **Langchain**, **HuggingFace Embeddings**, **FAISS**, **Ollama** and then **Streamlit** fot the UI.

Users are able to ask questions from documents uploaded to the specified folder (data).

---

## Setting Up
### Create a virtual environment
### Clone Github repo and Download Ollama
### Run ingest
### Run UI
